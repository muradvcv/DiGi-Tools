# 🛒 Digital Tools Marketplace

### Discover, Compare, and Purchase the Best Digital Tools for Productivity and Growth

A modern e-commerce platform for exploring and purchasing digital tools. Users can browse products, add them to their cart, manage selected items, and complete the checkout process through a clean and responsive user interface.

## 🚀 Features

* 🛒 Add products to cart with real-time cart count updates
* ❌ Remove individual products from the cart instantly
* 💳 Proceed to checkout functionality with cart reset
* 🔔 Interactive notifications using React Toastify
* 📱 Fully responsive design for mobile, tablet, and desktop
* 📦 Dynamic product rendering from JSON data
* 🎨 Modern and user-friendly UI based on the provided design

---

## 🛠️ Technologies Used

### Frontend

* React.js
* JavaScript (ES6+)
* Tailwind CSS
* DaisyUI

### State Management

* React Hooks (useState)

### Notifications

* React Toastify

### Data Source

* JSON

### 🛍️ Product Management
